Xatkit-SPL API

Xatkit-SPL Backend provides an API for chatbot modeling through different technologies. The main goal is to improve the utility of chatbots making them more accessible to the public with our automatic chatbot builder tool.

Intended Workflow

  1. User logs-in the website, creating a new user.
  2. Navigates to the configurator, and start defining a product-line-based chatbot family in an easy and understandable way + providing some basic information.
  3. Once the user has finished the modelling, hit enter and you will be prompted to create the chatbots you need.
  4. Define the name, the description and the intent information for this new chatbot product.
  5. Now, your chatbot is being tested with FLAMA, which is an automated tool for feature model analysis.
  6. Once everything is ready, the chatbot gets created through Xatkit, compiles all the necessary packages and gets automatically deployed in the localhost through Docker.

There are a few known bugs that we acknowledge, described in the projects section. If you detect any other new bug, please consider reporting it!
